# PHP本格じゃんけんゲーム (Janken Quest)

セッション（Session）機能を用いた状態管理を学び、実装するためのシンプルなWebベースのじゃんけんゲームです。1つのPHPファイルのみで「タイトル」「ゲーム」「ゲームクリア」「ゲームオーバー」の4つの画面遷移（ステート管理）を実現しています。

## 🎮 ゲームルール
- **タイトル画面**から「ゲームを始める」を押すと勝負がスタートします。
- プレイヤーは「✊」「✌️」「✋」のいずれかを選択してCOMと対戦します。
- **勝利条件**: 先に **3勝** するとゲームクリア画面へ遷移します。
- **敗北条件**: 先に **3敗** するとゲームオーバー画面へ遷移します。
- クリア、またはオーバー画面からいつでもタイトルに戻ってリセット可能です。

## 🛠️ 技術スタック & 仕組み
- **言語**: PHP 7.0 以上 / HTML5 / CSS3
- **状態管理**: `session_start()` を利用して、ユーザーのブラウザセッションに現在の画面（`$_SESSION['screen']`）や通算勝敗数（`$_SESSION['wins']` / `$_SESSION['losses']`）を安全に保持。
- **画面遷移**: POSTリクエストを受け取った後、PHPの条件分岐（`if-elseif`）によって動的に表示するHTMLブロックを切り替える「シングルページアプリケーション（SPA）」風の構造。

## 🚀 動かし方（ローカル環境）

### 1. 動作環境の準備
PHPが動く環境（XAMPP、MAMP、またはPHPのビルトインサーバーなど）を用意してください。

### 2. ファイルの配置
任意の公開フォルダ（例: `htdocs` や `www`）に、ソースコードを `index.php` という名前で保存します。

### 3. サーバーの起動とアクセス
